Jensen's Inequality

Convexity turns averaging into a powerful bound for optimization and machine learning.

Statement

Jensen's Inequality

If \(f\) is convex and \(X\) is an integrable random variable, then

\[f(\mathbb{E}[X]) \le \mathbb{E}[f(X)].\]

For discrete weights \(\theta_i \ge 0\), \(\sum_i \theta_i = 1\), this is equivalently

\[f\!\left(\sum_{i=1}^m \theta_i x_i\right) \le \sum_{i=1}^m \theta_i f(x_i).\]

Intuition

Convex functions curve upward. Taking an average input first and then evaluating \(f\) gives a value below evaluating \(f\) at each point and averaging afterward.

One-line Memory Hook

For convex \(f\): function of average is less than or equal to average of function.

Example 1: Quadratic Function

Let \(f(x)=x^2\), \(X\in\{-1,2\}\) with equal probability. Then

\[\mathbb{E}[X]=0.5,\quad f(\mathbb{E}[X])=0.25.\]

\[\mathbb{E}[f(X)] = \tfrac{1}{2}(1+4)=2.5.\]

So \(0.25 \le 2.5\), exactly matching Jensen's inequality.

Interactive: Jensen Check with Two-Point Distribution

Choose two support points \(x_1, x_2\) and weight \(\theta\in[0,1]\). We use \(f(x)=x^2\), compute \(f(\theta x_1+(1-\theta)x_2)\) and \(\theta f(x_1)+(1-\theta)f(x_2)\), and verify Jensen numerically.

Example 2: Why Variance is Nonnegative

Using \(f(x)=x^2\), Jensen gives

\[(\mathbb{E}[X])^2 \le \mathbb{E}[X^2].\]

Rearranging yields

\[\operatorname{Var}(X)=\mathbb{E}[X^2]-(\mathbb{E}[X])^2 \ge 0.\]

This is a classic and useful consequence that appears constantly in optimization and statistics.

Optimization Connections

  • Risk minimization: expected loss bounds from convex surrogates.
  • Mini-batch training: average-gradient and average-loss reasoning.
  • Convex relaxations: expectation-based upper and lower bounds.
  • Subgradient methods: links between averaging iterates and guarantees.
