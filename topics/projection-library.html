Projection Library for Convex Sets

Reusable projection operators for box, ball, simplex, and affine constraints.

Why This Page Matters

Many first-order methods reduce to a gradient-like step followed by a projection. Once projection operators are available, constrained optimization becomes practical across many model families.

\[x_{k+1} = \Pi_C\left(x_k - \alpha_k g_k\right).\]

Common Projection Operators

1) Box Set

For \(C=[-\rho,\rho]^n\), projection is coordinate-wise clipping: \(\Pi_C(v)_i=\max(-\rho,\min(\rho,v_i))\).

2) Euclidean Ball

For \(C=\{x:\|x\|_2\le r\}\), \(\Pi_C(v)=v\) if \(\|v\|_2\le r\), else \(\Pi_C(v)=r\,v/\|v\|_2\).

3) Simplex

For \(\Delta=\{x\ge 0,\;\sum_i x_i=1\}\), projection uses thresholding: \(x_i=\max(v_i-\tau,0)\) with \(\sum_i x_i=1\).

4) Affine Set

For \(C=\{x:Ax=b\}\), if \(AA^T\) is invertible: \(\Pi_C(v)=v-A^T(AA^T)^{-1}(Av-b)\).

Interactive: 2D Projection Explorer

Usage Checklist

  • Match each constraint type to its projection formula first.
  • Test numerical stability near boundaries.
  • Reuse projection operators across PGD, proximal, and ADMM variants.
  • Profile projection cost if constraints become high-dimensional.
