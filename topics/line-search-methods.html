Line Search Methods

Choose step sizes adaptively using sufficient decrease and curvature criteria.

Why Line Search?

Gradient methods need a step size \(\alpha_k\). A fixed value can be too small (slow) or too large (unstable). Line search computes \(\alpha_k\) from local objective and gradient information.

Armijo Backtracking

For descent direction \(p_k\), accept \(\alpha\) when

\[f(x_k+\alpha p_k) \le f(x_k) + c_1\alpha\nabla f(x_k)^Tp_k,\]

with \(c_1\in(0,1)\). Start from \(\alpha_0\), shrink by \(\rho\in(0,1)\) until the condition holds. 

alpha = alpha0
while f(x + alpha p) > f(x) + c1 * alpha * grad_f(x)^T p:
    alpha = rho * alpha
return alpha

Wolfe Conditions

Strong Wolfe adds a curvature condition to avoid steps that are too short:
