Subgradients & Optimality

Handle non-smooth convex functions and certify minimizers with first-order conditions.

Why Subgradients?

Many useful convex objectives are not differentiable everywhere. Examples include absolute value, the L1 norm, hinge loss, and max-of-affine functions. Gradients are unavailable at corners, but subgradients let us keep first-order reasoning and algorithms.

Typical Non-smooth Models

  • Lasso: \(\min_x \frac{1}{2}\|Ax-b\|_2^2 + \lambda\|x\|_1\)
  • Robust regression: \(\min_x \|Ax-b\|_1\)
  • SVM hinge objective: max-based piecewise linear loss

Definition: Subgradient and Subdifferential

Subgradient

Let \(f : \mathbb{R}^n \to \mathbb{R}\) be convex. A vector \(g\) is a subgradient of \(f\) at \(x\) if for every \(y\),

\[f(y) \ge f(x) + g^T(y-x).\]

The set of all such vectors is the subdifferential, denoted \(\partial f(x)\).

Geometrically, each subgradient defines a supporting hyperplane that lies below the graph of \(f\) and touches it at \(x\).

Core Optimality Condition

For unconstrained convex minimization \(\min_x f(x)\), the first-order condition is:

\[x^* \text{ is optimal } \Longleftrightarrow 0 \in \partial f(x^*).\]

This extends the smooth rule \(\nabla f(x^*) = 0\). For constrained problems \(\min_{x \in C} f(x)\), an equivalent condition is \(0 \in \partial f(x^*) + N_C(x^*)\), where \(N_C(x^*)\) is the normal cone.

Worked Example: \(f(x)=|x|\)

Subdifferential of Absolute Value

\[ \partial |x| = \begin{cases} \{1\}, & x > 0, \\ [-1,1], & x = 0, \\ \{-1\}, & x < 0. \end{cases} \]

Since \(0 \in [-1,1] = \partial |0|\), \(x^*=0\) is optimal. This is the simplest non-smooth certificate of global optimality.

Worked Example: Soft-Thresholding

Consider one-dimensional L1-regularized least squares:

\[\min_x \frac{1}{2}(x-v)^2 + \lambda |x|.\]

The optimal solution is the soft-threshold map:

\[x^* = \operatorname{sign}(v)\max(|v|-\lambda,0).\]

Interpretation

Values with \(|v| \le \lambda\) collapse to zero, while larger values are shrunk toward zero. This is why L1 regularization promotes sparsity.

Practice Checklist

  • Identify where the function is non-smooth.
  • Write \(\partial f(x)\) explicitly at smooth and corner points.
  • Use \(0 \in \partial f(x^*)\) to certify minimizers.
  • Connect this to proximal methods and projected/subgradient algorithms.
