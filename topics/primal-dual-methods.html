Primal-Dual Methods (PDHG / Chambolle-Pock)

Update primal and dual variables together to solve structured convex composite objectives efficiently.

Problem Template

Primal-dual methods are especially effective for saddle formulations such as

\[\min_x\; f(x) + g(Kx),\]

where \(f\) and \(g\) are convex and proximable, and \(K\) is a linear operator.

Saddle-Point View

Introduce dual variable \(y\) via conjugacy:

\[\min_x\max_y\; f(x) + \langle Kx,y\rangle - g^*(y).\]

Alternating updates on \(x\) and \(y\) produce scalable first-order algorithms with simple proximal substeps.

PDHG / Chambolle-Pock Iteration

A common variant is

\[\begin{aligned} y^{k+1} &= \operatorname{prox}_{\sigma g^*}(y^k + \sigma K\bar{x}^k),\\ x^{k+1} &= \operatorname{prox}_{\tau f}(x^k - \tau K^Ty^{k+1}),\\ \bar{x}^{k+1} &= x^{k+1} + \theta(x^{k+1}-x^k). \end{aligned}\]

A standard stability condition is \(\tau\sigma\|K\|^2 < 1\). 

repeat
    y <- prox_{sigma g*}( y + sigma * K * x_bar )
    x <- prox_{tau f}( x - tau * K^T * y )
    x_bar <- x + theta * (x - x_prev)
until residuals are small

What to Monitor

  • Primal residual for feasibility/prox consistency.
  • Dual residual for stationarity progress.
  • Primal-dual gap proxy for stopping decisions.
