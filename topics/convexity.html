Understanding Convexity

The foundation of convex optimization

What is a Convex Set?

A convex set is one of the most fundamental concepts in convex optimization. Understanding convex sets is essential because they define the feasible regions where optimization problems can be efficiently solved.

Definition: Convex Set

A set \(C \subseteq \mathbb{R}^n\) is called convex if for any two points \(x, y \in C\) and any \(\theta\) with \(0 \leq \theta \leq 1\), the point

\[\theta x + (1-\theta)y \in C\]

In other words, a set is convex if the line segment connecting any two points in the set lies entirely within the set.

Intuitive Understanding

Imagine drawing a straight line between any two points in a shape. If that line always stays inside the shape (no matter which two points you choose), then the shape is convex. If the line ever goes outside the shape, it's not convex.

Left: a convex set (disk). Right: a non-convex set (crescent shape) where some line segments between points leave the set.

Example: The Unit Ball

Example 1: Unit Ball in \(\mathbb{R}^2\)

Consider the unit ball (disk) in two dimensions:

\[C = \{(x_1, x_2) \in \mathbb{R}^2 : x_1^2 + x_2^2 \leq 1\}\]

This set is convex. To see why, take any two points \((x_1, x_2)\) and \((y_1, y_2)\) in the unit ball, meaning \(x_1^2 + x_2^2 \leq 1\) and \(y_1^2 + y_2^2 \leq 1\).

For any \(0 \leq \theta \leq 1\), consider the point:

\[z = \theta(x_1, x_2) + (1-\theta)(y_1, y_2)\]

Using the triangle inequality and properties of norms, we can show that \(\|z\|_2 \leq \theta\|x\|_2 + (1-\theta)\|y\|_2 \leq 1\), which means \(z\) is also in the unit ball. Therefore, the unit ball is convex.

Any two points in the unit ball can be joined by a line segment that stays entirely inside the ball, so the set is convex.

Properties of Convex Sets

Convex sets have several important properties that make them useful in optimization:

  • The intersection of convex sets is convex
  • Any affine transformation of a convex set is convex
  • The sum (Minkowski sum) of convex sets is convex
  • A local minimum in a convex set is also a global minimum

Common Examples

Here are some common examples of convex sets:

  • Hyperplanes: \(\{x : a^T x = b\}\)
  • Halfspaces: \(\{x : a^T x \leq b\}\)
  • Balls: \(\{x : \|x - x_0\| \leq r\}\)
  • Polyhedra: \(\{x : Ax \leq b\}\)
  • Ellipsoids: \(\{x : (x-x_0)^T P^{-1} (x-x_0) \leq 1\}\) where \(P\) is positive definite
