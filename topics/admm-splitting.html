ADMM

Split complex convex objectives into simpler subproblems with strong practical scalability.

Problem Form

ADMM targets objectives that can be split into two blocks:

\[\min_{x,z} \; f(x) + g(z) \quad \text{subject to } Ax + Bz = c.\]

This covers many sparse learning, constrained estimation, and distributed optimization problems.

Augmented Lagrangian

ADMM uses the augmented Lagrangian

\[\mathcal{L}_\rho(x,z,y) = f(x)+g(z)+y^T(Ax+Bz-c)+\frac{\rho}{2}\|Ax+Bz-c\|_2^2.\]

The quadratic penalty stabilizes updates and improves practical convergence behavior.

Scaled ADMM Updates

With scaled dual variable \(u=y/\rho\), a common iteration is: 

repeat
    x^{k+1} = argmin_x f(x) + (rho/2)||Ax + Bz^k - c + u^k||_2^2
    z^{k+1} = argmin_z g(z) + (rho/2)||Ax^{k+1} + Bz - c + u^k||_2^2
    u^{k+1} = u^k + Ax^{k+1} + Bz^{k+1} - c
until residuals are small

Residual Diagnostics

Stopping Criteria

Track primal residual \(r^k = Ax^k+Bz^k-c\) and dual residual \(s^k = \rho A^TB(z^k-z^{k-1})\) (form depends on scaling). Stop when both are below tolerances.

Balancing these residuals is often more informative than monitoring objective value alone.

Interactive: ADMM Residual Calculator (Scalar Split)

Use the scalar split constraint \(x-z=c\) (equivalent to \(Ax+Bz=c\) with \(A=1, B=-1\)). Adjust \(x^k, z^k, z^{k-1}, c, \rho\), and inspect primal residual \(r^k=x^k-z^k-c\) and dual residual \(s^k=\rho(z^k-z^{k-1})\).

Worked Template: Lasso via Splitting

Lasso can be written as

\[\min_{x,z} \frac{1}{2}\|Ax-b\|_2^2 + \lambda\|z\|_1 \quad \text{s.t. } x-z=0.\]

Then ADMM alternates a ridge-like \(x\)-update, soft-thresholding \(z\)-update, and a simple dual ascent step.

When ADMM Works Well

  • Each block subproblem is easy or has a closed form prox.
  • Data and variables can be split across machines.
  • You need robust behavior on large sparse problems.
  • Primal-dual residual monitoring is natural for your constraints.
