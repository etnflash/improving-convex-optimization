Interior-Point Methods

Solve constrained convex programs by following the central path through the interior.

Target Problem

\[\min_x \; f_0(x) \quad \text{s.t. } f_i(x) \le 0,\; i=1,\dots,m,\; Ax=b.\]

Interior-point methods replace inequality constraints with a smooth barrier and solve a sequence of unconstrained (or equality-constrained) subproblems.

Log-Barrier Formulation

Define the barrier

\[\phi(x) = -\sum_{i=1}^m \log\big(-f_i(x)\big),\]

valid only in the strict interior \(f_i(x)<0\). Then solve

\[\min_x \; t f_0(x) + \phi(x) \quad \text{s.t. } Ax=b,\]

where \(t\) increases over outer iterations to approach the original constrained optimum.

Central Path and Duality Gap

Solutions \(x^*(t)\) define the central path. As \(t\to\infty\), \(x^*(t)\) approaches the true solution.

A standard estimate of suboptimality is

\[\text{gap} \approx \frac{m}{t}.\]

This gives a principled stopping rule for the outer loop.

Newton Step (Inner Solve)

Each barrier subproblem is solved using Newton's method with line search, typically through KKT systems. 

repeat (outer)
    approximately solve barrier subproblem at current t using Newton steps
    t <- mu * t    (mu > 1)
until m / t <= epsilon

Practical Defaults

Common choices: \(\mu\in[10,20]\), backtracking line search, and strict-feasibility safeguards.

Why It Is Powerful

  • High accuracy for medium-to-large convex programs.
  • Strong theory and robust behavior on LP/QP/SOCP/SDP families.
  • Primal-dual variants provide excellent stopping diagnostics.
  • Widely used in mature optimization solvers.

Worked Mini Case: Linear Program

For \(\min c^Tx\) s.t. \(Ax=b,\; x\ge 0\), the barrier subproblem is

\[\min_x \; t c^T x - \sum_i \log x_i \quad \text{s.t. } Ax=b.\]

Newton directions are computed from linearized KKT equations, and line search keeps \(x\) positive.
